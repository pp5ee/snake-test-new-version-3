# 贪吃蛇大作战 - 项目规格说明书

## 1. 项目概述

- **项目名称**: 贪吃蛇大作战 (Snake Battle)
- **项目类型**: Web游戏 (React单页应用)
- **核心功能**: 多人对战风格的贪吃蛇游戏，玩家与AI蛇对抗，支持吃蛇机制，分数本地持久化
- **目标用户**: 休闲游戏玩家

## 2. UI/UX 规格

### 布局结构

```
+------------------+------------------------+
|                  |                        |
|    左侧面板       |       游戏主区域        |
|   (蛇选择器)      |     (Canvas画布)       |
|                  |                        |
|   - 蛇类型选择    |                        |
|   - 历史记录      |                        |
|   - 控制按钮      |                        |
|                  |                        |
+------------------+------------------------+
```

- **左侧面板宽度**: 280px
- **游戏区域**: 自适应，最小 600x600px
- **响应式**: 断点 768px（移动端纵向布局）

### 视觉设计 - 赛博朋克风格

**色彩方案**:
- 背景主色: `#0a0a0f` (深黑蓝)
- 背景次色: `#12121a` (暗紫黑)
- 主强调色: `#00fff5` (霓虹青)
- 次强调色: `#ff00ff` (霓虹粉)
- 警告色: `#ff3366` (霓虹红)
- 成功色: `#00ff66` (霓虹绿)
- 文字主色: `#e0e0e0`
- 文字次色: `#888899`

**渐变与发光**:
- 蛇身: 线性渐变 + 霓虹发光 (box-shadow/filter: drop-shadow)
- 食物: 脉冲发光动画
- 网格背景: 半透明线条网格
- 边框: 霓虹发光边框

**字体**:
- 标题: `"Orbitron", sans-serif` (Google Fonts)
- 正文: `"Rajdhani", sans-serif` (Google Fonts)
- 数字/分数: `"Orbitron"` 加粗

**间距系统**:
- 基础单位: 8px
- 组件间距: 16px / 24px
- 内边距: 12px / 16px / 24px

### 组件列表

1. **蛇选择器** (SnakeSelector)
   - 6种蛇类型可选，每种有独特配色和纹理
   - 选中状态: 霓虹发光边框 + 缩放动画
   - 类型: 经典青蛇、霓虹粉蛇、毒液绿蛇、熔岩红蛇、冰霜蓝蛇、黄金蛇

2. **游戏画布** (GameCanvas)
   - Canvas渲染，网格背景
   - 玩家蛇、AI蛇、食物的渲染
   - 动态效果层（粒子、特效）

3. **控制面板** (ControlPanel)
   - 开始/暂停按钮（霓虹风格切换）
   - 重新开始按钮
   - 速度控制滑块

4. **分数面板** (ScorePanel)
   - 当前分数（大型数字显示）
   - 最高分显示
   - 历史记录列表（最近10场）

5. **游戏状态Overlay**
   - 暂停时显示半透明遮罩 + "PAUSED" 文字
   - 游戏结束时显示结果（胜利/失败）+ 分数统计

### 动态效果

1. **蛇身发光**: 每节蛇身有呼吸灯效果
2. **食物脉冲**: 食物有缩放+发光脉冲动画
3. **吃食物特效**: 粒子爆炸效果
4. **吃蛇特效**: 大范围粒子爆发 + 屏幕震动
5. **死亡特效**: 蛇身闪烁后消散
6. **背景网格**: 缓慢移动的扫描线效果
7. **UI动画**: 按钮 hover/click 霓虹闪烁
8. **分数变化**: 数字增加时有滚动动画

## 3. 功能规格

### 核心游戏机制

**玩家控制**:
- 方向键 / WASD 控制移动方向
- 空格键 暂停/继续
- 移动后不可反向（180度掉头禁止）

**AI蛇行为**:
- 初始数量: 5条
- 随机位置生成，避免与玩家重叠
- 简单AI: 随机方向移动，避开边界和自身
- 吃到食物或吃蛇后增长

**吃蛇规则**:
- 蛇头接触对方蛇身可吃掉对方
- 规则: 长度长的吃掉长度短的（长度相同随机）
- 吃蛇后获得对方长度×10的分数
- 吃蛇后自身长度增加对方长度的一半

**食物机制**:
- 场上最多10个食物
- 吃掉后随机位置刷新
- 吃食物 +10 分，长度+1

**死亡判定**:
- 撞墙死亡
- 撞自己死亡
- 被比自己长的蛇吃掉死亡

**游戏流程**:
1. 选蛇类型 → 2. 点击开始 → 3. 玩家移动吃食物/吃蛇 → 4. 死亡 → 5. 显示结果 → 6. 记录分数 → 7. 可重新开始

### 数据持久化

**localStorage存储**:
```json
{
  "snakeBattle": {
    "highScore": 1250,
    "history": [
      { "score": 1250, "date": "2024-01-15T10:30:00Z", "snakeType": "neon-pink" },
      { "score": 800, "date": "2024-01-15T09:15:00Z", "snakeType": "classic" }
    ]
  }
}
```

- 最多保留最近20条历史记录
- 超过时删除最旧的
